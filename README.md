# 🌍 Air Quality Analyzer

An interactive **Air Quality Analysis Dashboard** built with **Streamlit**, **Pandas**, and **Plotly**. The application analyzes global air quality data, calculates Air Quality Index (AQI), visualizes pollution trends, compares countries, and displays pollution levels on an interactive world map.

## 🚀 Features

- 📊 Country-wise Air Quality Analysis
- 🌫️ Multi-Pollutant AQI Calculation
- 📈 Monthly Pollution Trend Analysis
- 🥧 Pollutant Distribution Charts
- 🔥 Pollutant Correlation Heatmap
- 🌍 Global Pollution Ranking
- ⚖️ Country-to-Country Pollution Comparison
- 🗺️ Interactive Global Pollution Map
- 🎨 Clean and Interactive Streamlit Dashboard

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Air-Purifier.git
```

Navigate to the project:

```bash
cd Air-Purifier
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
